개인정보 처리방침 (Privacy Policy)

본 개인정보 처리방침은 [social_archive](이하 "social_archive 서비스")가 사용자의 개인정보를 어떻게 수집, 사용 및 보호하는지 설명합니다. 본 서비스는 TikTok for Developers 플랫폼의 API 서비스를 이용하며, TikTok의 정책을 준수합니다.

1. 수집하는 개인정보의 항목

서비스는 TikTok 계정을 통한 로그인 및 서비스 제공을 위해 다음과 같은 정보를 TikTok API로부터 제공받아 처리합니다:

  • 기본 프로필 정보 (user.info.basic, user.info.profile): 사용자 고유 ID(Open ID), 닉네임, 프로필 사진.
  • 계정 통계 정보 (user.info.stats): 팔로워 수, 팔로잉 수, 좋아요 수, 동영상 개수 등의 공개된 통계 데이터.
  • 동영상 데이터 (video.list): 사용자가 업로드한 동영상 목록, 동영상 커버 이미지, 재생 수 및 관련 메타데이터.

2. 개인정보의 수집 및 이용 목적

수집된 정보는 오직 다음의 목적을 위해서만 사용됩니다:

  • 서비스 로그인 및 식별: TikTok 계정을 연동하여 사용자를 식별하고 로그인 상태를 유지하기 위함.
  • 기능 제공: 사용자의 TikTok 활동 통계를 시각화하여 보여주거나, 동영상 목록을 아카이빙 및 관리하는 기능을 제공하기 위함.
  • 공유 기능 (Share Kit): 사용자가 서비스 내의 콘텐츠를 TikTok으로 다시 공유할 수 있도록 지원하기 위함.
  • 실시간 업데이트 (Webhooks): 사용자 데이터 변경 사항을 실시간으로 감지하여 최신 상태를 유지하기 위함.

3. 개인정보의 제3자 제공 및 공유

서비스는 사용자의 동의 없이 개인정보를 외부에 공개하거나 제3자에게 판매하지 않습니다. 단, 법령에 의한 요구가 있거나, 서비스 제공을 위해 필수적인 기술적 처리가 필요한 경우(예: 클라우드 서버 저장)는 예외로 합니다.

4. 데이터의 보유 및 파기

사용자의 개인정보는 서비스 이용 기간 동안 보유하며, 사용자가 서비스 탈퇴를 요청하거나 TikTok 설정에서 앱 연결을 해제할 경우 해당 데이터는 지체 없이 파기됩니다.

5. 사용자의 권리 (데이터 삭제 요청)

사용자는 언제든지 TikTok 앱의 [설정 > 보안 및 로그인 > 앱 권한] 메뉴에서 본 서비스의 접근 권한을 철회할 수 있습니다. 권한 철회 시 서비스는 더 이상 사용자의 데이터에 접근하지 않으며, 저장된 데이터 삭제를 원하실 경우 문의 주시면 즉시 조치하겠습니다.
